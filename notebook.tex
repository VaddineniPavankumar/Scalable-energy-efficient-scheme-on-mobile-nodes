    \subsubsection{we need}\label{we-need}

N - the total number of HN nodes

Fs - scalability factor which define as the maximum number of nodes to
be accommodated in a single zone

L - length of area

B - cardinality of local base station(LBS)

n - number of heterogeneity level

If N is the total number of nodes and n is the number of heterogeneity
levels, then the network has n categories (types) of nodes: cate-1,
cate-2, cate-3, ..., cate-n nodes with their cardinalities: N1t , N2t ,
N3t , ..., Nnt and energy levels: E1t , E2t , E3t , ..., Ent
respectively, where:

\begin{verbatim}
                        N1t > N2t > N3t > ⋯ > Nnt
                        
                        E1t < E2t < E3t < ⋯ < Ent
                        
\end{verbatim}

The different levels of node energies are related according to the
following formula, where the initial energy of cate-i nodes is alpha
times more than that of cate-(i − 1):

\begin{verbatim}
                        Eit = E1t × (1 + (i − 1) × aplha)
                        
\end{verbatim}

where alpha is constant. The number of cate-i nodes in the network is
given by:

\begin{verbatim}
                        Nit = N × (beta − gamma1) × (beta − gamma2) × (beta − gamma3)×⋯× (beta − gammai)
                        
                        N1t + N2t + N3t + ⋯ + Nnt  = N
                        
\end{verbatim}

alpha and betai are the primary and secondary parameters of
heterogeneity in the Singh model respectively (Singh 2017), and are
related according to the following formula:

\begin{verbatim}
        (beta − gamma1) × (1 + (beta − gamma2) × (1 + (beta − gamma3)×⋯× (1 + (beta − gamman)))) = 1
\end{verbatim}

and gammai is given by:

\begin{verbatim}
                        gammai = gammai−1 − 2 × theta
\end{verbatim}

where Θ(theta) is constant. The value of first secondary parameter
gamma1 is predefined and must satisfy the following inequality:

\begin{verbatim}
                        gamma1 / (2 × (n − 1)) > Θ
\end{verbatim}
